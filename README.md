# Проект "Параметры"

Проект представляет собой веб-приложение для управления системными и пользовательскими параметрами. Цель — создать интуитивно понятный интерфейс для просмотра, фильтрации и редактирования различных конфигурационных настроек.

Приложение разрабатывается с использованием HTML, CSS и JavaScript. Данный репозиторий используется в учебных целях для моделирования процесса разработки по методологии Scrum.

В рамках лабораторной работы мы имитируем один спринт, используя GitHub Issues для управления бэклогом продукта и отслеживания прогресса.
